Smart Parking API

API pour la prédiction de durée et d'occupation de stationnement

Documentation de l'API

Bienvenue sur la documentation de l'API Smart Parking. Cette API permet de prédire la durée de stationnement et l'occupation des places de parking.

GET /health

Vérifier l'état de l'API et si les modèles sont correctement chargés.

Réponse

{
    "status": "ok",
    "message": "API is running and models are loaded"
}

GET /features

Obtenir la liste des caractéristiques nécessaires pour les prédictions.

Réponse

{
    "regression_features": ["feature1", "feature2", "feature3", "feature4"],
    "classification_features": ["feature1", "feature2", "feature3"]
}

POST /predict/duration

Prédire la durée de stationnement.

Corps de la requête

{
    "feature1": value1,
    "feature2": value2,
    "feature3": value3,
    "feature4": value4
}

Réponse

{
    "prediction": 2.5,
    "unit": "hours"
}

POST /predict/occupancy

Prédire l'occupation d'une place de parking.

Corps de la requête

{
    "feature1": value1,
    "feature2": value2,
    "feature3": value3
}

Réponse

{
    "prediction": 1,
    "probability": {
        "not_occupied": 0.15,
        "occupied": 0.85
    },
    "label": "Occupied"
}

Utilisation avec le frontend React

Cette API est conçue pour être utilisée avec un frontend React qui offre une interface utilisateur conviviale.

Pour lancer le frontend, accédez au dossier /frontend et exécutez les commandes suivantes :

npm install
npm start
